# Docker 입문자를 위한 세미나

## 안내 사항
- 가상화를 지원하지 않는 CPU의 경우 도커를 사용 할 수 없습니다.

- Windows10 Edu 이하 버전을 사용하는 경우 Docker for Windows를 사용할 수 없습니다. Docker Toolbox를 설치해주세요.
